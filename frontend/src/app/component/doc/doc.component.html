Documentation de l'Application Tracking GPS

Bienvenue dans la documentation de l'application Tracking GPS développée par Lucas Pauzies. Cette application simule l'IoT d'un système de suivi GPS en temps réel, utilisant une architecture microservices.

Objectif du Projet

L'objectif principal de ce projet est de créer une application de suivi GPS avec l'affichage en temps réel de la position des machines sur une carte. L'architecture microservices mise en place consiste en un broker Kafka accessible depuis deux machines physiques (IP1, IP2), deux producteurs Kafka générant des coordonnées GPS cohérentes, un consommateur Kafka stockant ces informations dans une base de données PostgreSQL, et une API FastAPI fournissant des routes pour récupérer les données en temps réel. Un frontend utilisant une librairie de cartographie (par exemple, Leaflet) affiche la position des machines sur la carte.
